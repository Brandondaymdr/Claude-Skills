# Keyboard Shortcuts Reference

## Essential Shortcuts (Learn These First)

These are the shortcuts that separate fast editors from slow ones. If you learn nothing else,
learn these:

| Action | Mac | Windows | Why it matters |
|--------|-----|---------|---------------|
| Play/Pause | Space | Space | Basic playback |
| Play backward | J | J | Shuttle review |
| Stop | K | K | Shuttle stop |
| Play forward | L | L | Shuttle review |
| Mark In | I | I | Set start of selection |
| Mark Out | O | O | Set end of selection |
| Insert edit | , (comma) | , (comma) | Add clip pushing right |
| Overwrite edit | . (period) | . (period) | Add clip replacing |
| Cut at playhead | Cmd+K | Ctrl+K | Split clip instantly |
| Ripple trim previous | Q | Q | Remove left side to playhead |
| Ripple trim next | W | W | Remove right side to playhead |
| Undo | Cmd+Z | Ctrl+Z | Fix mistakes |
| Ripple Delete | Shift+Delete | Shift+Delete | Delete and close gap |
| Lift | ; (semicolon) | ; (semicolon) | Remove clip, leave gap |
| Extract | ' (apostrophe) | ' (apostrophe) | Remove clip, close gap |

### The Q and W Shortcuts (Game Changers)
These two shortcuts are worth emphasizing because they transform editing speed:

**Q (Ripple Trim Previous to Playhead)**: Everything to the LEFT of the playhead on the
selected clip gets removed, and the timeline closes the gap. Park your playhead where the
good part starts, press Q, and the fluff before it disappears.

**W (Ripple Trim Next to Playhead)**: Everything to the RIGHT of the playhead on the
selected clip gets removed, and the timeline closes the gap. Park where the good part ends,
press W, and the rest vanishes.

Combined workflow: Play with J/K/L, find where the good part starts, press Q. Play forward,
find where it ends, press W. Done. This is the core loop of fast editing.

## Customizing Shortcuts

1. **Edit > Keyboard Shortcuts** (or Cmd/Ctrl+Opt/Alt+K)
2. Search for any command by name
3. Click in the shortcut field and press your desired keys
4. Premiere warns you if there's a conflict
5. Save as a custom preset to transfer between machines

### Popular Custom Shortcuts
Many editors customize these for faster editing:
- **Zoom to Sequence** mapped to something easy (default \ is awkward on some keyboards)
- **Add Edit to All Tracks** — useful for multicam workflows
- **Previous/Next marker** — useful for marker-based workflows
- **Toggle Full Screen** — for presentations/reviews

### Transferring Shortcuts
Keyboard shortcuts are saved as .kys files. To transfer:
1. Edit > Keyboard Shortcuts > Save As (name your layout)
2. Find the .kys file in your preferences folder
3. Copy to the same location on the new machine
4. Edit > Keyboard Shortcuts > load the preset

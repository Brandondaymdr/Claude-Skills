# Toast Web Navigation Reference

Complete guide to finding every setting and feature in Toast Web (pos.toasttab.com).

## Accessing Toast Web
- URL: `https://pos.toasttab.com`
- Click "Login" in the upper right
- Enter email and password
- Select your restaurant location if you have multiple

---

## Main Navigation Sections

### Home / Dashboard
**Path:** Home (default landing page after login)
- Quick view of today's sales, order count, and key metrics
- Recent activity feed
- Quick links to common tasks

### Orders
**Path:** Orders
- **Orders Hub**: View all current and recent orders (dine-in, online, delivery)
- Filter by: status, order source, date range
- Click any order to view full details, modify, or refund

### Menu Management
**Path:** Menu Management
- **Menus**: Create/edit menus, menu groups, and items
  - Menu Management > Menus > [Select Menu] > [Edit]
- **Menu Groups**: Organize items into categories
- **Menu Items**: Individual items with prices, descriptions, modifiers
  - Menu Management > Menus > [Menu] > [Group] > [Item]
- **Modifier Groups**: Customization options (sizes, add-ons, etc.)
  - Menu Management > Modifier Groups
- **Publishing**: Push menu changes to POS devices
  - Menu Management > Publishing > Publish All

### Online Ordering
**Path:** Online Ordering
- **Settings**: Enable/disable, configure branding and appearance
- **Schedules**: Set online ordering hours (can differ from dine-in)
- **Menu**: Choose which items appear online
- **Fulfillment**: Prep times, throttling, delivery settings

### Reporting
**Path:** Reporting
- **Sales Summary**: Reporting > Sales > Sales Summary
- **Product Mix**: Reporting > Sales > Product Mix
- **Hourly Sales**: Reporting > Sales > Hourly Sales
- **Payment Summary**: Reporting > Payments > Payment Summary
- **Tips Report**: Reporting > Payments > Tips
- **Labor Summary**: Reporting > Labor > Labor Summary
- **Employee Performance**: Reporting > Labor > Employee Performance
- **Void Report**: Reporting > Sales > Voids
- **Discount Report**: Reporting > Sales > Discounts
- **Cash Report**: Reporting > Payments > Cash
- **Guest Report**: Reporting > Guests
- **Export**: Most reports allow CSV/Excel export

### Employees
**Path:** Employees
- **Employee List**: Employees > Employees
  - Add Employee: Employees > Employees > Add
  - Edit Employee: Click on employee name
- **Jobs**: Employees > Jobs (define roles, wages, permissions)
- **Scheduling**: Employees > Scheduling (if enabled)
- **Permissions**: Set through Jobs â€” each job has permission levels
- **Tips**: Employees > Tips (configure tip pooling/distribution)

### Payments
**Path:** Setup > Payment (or Payments section)
- **Payment Types**: Configure which payment methods to accept
- **Service Charges**: Setup > Service Charges (auto-grat, delivery fees)
- **Tax Rates**: Setup > Tax Rates
- **Gift Cards**: Setup > Gift Cards (if using Toast gift cards)

### Kitchen
**Path:** Kitchen (or Setup > Kitchen)
- **Prep Stations**: Define stations (bar, grill, expo, etc.)
- **Routing**: Configure which items go to which station
- **KDS Settings**: Kitchen display system configuration
- **Printers**: Kitchen printer assignments

### Marketing
**Path:** Marketing
- **Loyalty**: Configure loyalty program
- **Email Marketing**: Create and send campaigns
- **Guest Data**: View and manage guest database

### Setup (General Configuration)
**Path:** Setup
- **General**: Restaurant name, address, contact info
  - Setup > General > Restaurant Info
- **Dining Options**: Dine In, Takeout, Delivery
  - Setup > Dining Options
- **Revenue Centers**: Bar, Patio, Main Dining, etc.
  - Setup > Revenue Centers
- **Hours**: Operating hours and schedules
  - Setup > Hours
- **Hardware**: Manage POS devices, printers, terminals
  - Setup > Hardware
- **Cash Management**: Starting amounts, drawer settings
  - Setup > Cash Management
- **Integrations**: Third-party service connections
  - Setup > Integrations

## Quick-Find Cheat Sheet

| I want to... | Go to... |
|---|---|
| Change a menu item price | Menu Management > Menus > [Menu] > [Group] > [Item] > Edit |
| Add a new employee | Employees > Employees > Add |
| Set up happy hour | Menu Management > Menus > Create new menu > Set availability hours |
| View today's sales | Reporting > Sales > Sales Summary |
| See what's selling | Reporting > Sales > Product Mix |
| Check labor costs | Reporting > Labor > Labor Summary |
| Set up online ordering | Online Ordering > Settings |
| Add a modifier (like milk options) | Menu Management > Modifier Groups > Create |
| Configure tip pooling | Employees > Tips |
| Set up a new prep station | Kitchen > Prep Stations > Add |
| 86 an item | Menu Management > [find item] > toggle Out of Stock |
| Export data | Reporting > [any report] > Export button (CSV/Excel) |
| Change tax rate | Setup > Tax Rates |
| Add a service charge | Setup > Service Charges |
| View open tabs | Orders (from POS) or Orders Hub (Toast Web) |
| Set employee permissions | Employees > Jobs > [Job] > Permissions |
| Configure nightly email report | Reporting > Settings > Email Reports |
